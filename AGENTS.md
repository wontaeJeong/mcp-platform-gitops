## CONVENTIONS

- Runtime resources use namespace `mcp-gateway`; namespace and live Secrets are out-of-band prerequisites.
- `argocd/root-application.yaml` is the deployment entry. It creates three fixed platform Applications and the root-managed MCP server ApplicationSet.
- The ApplicationSet requires the Argo CD ApplicationSet CRD and controller before the root can reconcile it.
- Root sync-wave `50` only orders ApplicationSet creation. It does not wait for generated MCP server Application health or readiness.
- Add MCP backends under `apps/mcp-servers/<name>/`; the generated Argo CD Application name is `<name>-mcp-server`.
- Removing an `apps/mcp-servers/<name>/` directory deletes the generated Application and prunes its workload resources.
- `REPLACE_WITH_GIT_SHA` is deliberate in gateway and mock kustomizations; external CI stamps source revisions.
- Kustomize generators own the bootstrap script and gateway configuration ConfigMaps. Edit generator inputs, not rendered names.
- Gateway signs `X-MCP-Identity` using `MCP_INTERNAL_JWT_SECRET`; mock verifies the same Secret using `MCP_IDENTITY_JWT_SECRET`.
- Contour and `star-mcp-aidev-tls` are production ingress assumptions. Gateway and Keycloak are public; mock is not.
- Local images are built from sibling repositories, pushed through `localhost:5000`, and consumed by immutable digest.
- Tests are shell executable specifications, not a framework suite; exact strings encode security and topology contracts.

## ANTI-PATTERNS (THIS PROJECT)

- Never commit live Secret values or a corporate CA bundle. Only `secrets.example.yaml` and `local/.env.example` are commit-safe templates.
- Never apply `secrets.example.yaml` or add it to Kustomize resources.
- Never bypass TLS with `curl -k` or `--insecure`; provide the real CA bundle through an absolute local path or Secret.
- Never expose `mock-mcp-server` through an Ingress or host port; gateway is the sole public MCP path.
- Never mask Keycloak bootstrap failures with `|| true`; retries are bounded and attachment failures are fatal.
- Never move MCP Inspector into Compose or give it the Docker socket/application secrets.
- Never replace digest-pinned local image references with mutable tags or Compose builds.

## UNIQUE STYLES

- Production is app-of-apps plus three fixed platform Applications and one MCP server ApplicationSet; there is no parent `apps/kustomization.yaml`.
- `apps/keycloak-bootstrap/bootstrap.sh` is generated into a ConfigMap and mounted into a hook Job.
- Dirty sibling worktrees receive a deterministic SHA-256 source identity; clean trees use the full Git commit SHA.
- `tests/local-compose.sh` embeds Python for structured JSON assertions and deliberately invokes other contract scripts.
